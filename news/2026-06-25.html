OpenAI 發表首款自研推論晶片 Jalapeño，Broadcom 代工打造

TechCrunch｜2026-06-24 22:54（UTC）｜✍️ Russell Brandom

OpenAI 正式推出旗下首款自研 AI 推論晶片「Jalapeño」，由 Broadcom 協助設計與製造。這款晶片專為 OpenAI 的推論工作負載量身打造，初期測試顯示效能功耗比顯著優於現有替代方案。晶片代號取自墨西哥辣椒，象徵其「小巧但強大」的特性。OpenAI 強調，晶片開發過程中亦使用自家 AI 模型輔助設計，目前仍處於測試階段。這次合作早在 2025 年 10 月便已對外宣布。

Jalapeño 的問世代表 AI 產業垂直整合的重要一步。過去 OpenAI 高度依賴 Nvidia GPU，如今自研晶片可望大幅降低推論成本。OpenAI 總裁 Greg Brockman 表示：「我們對工作負載有深入了解，一直在尋找哪些需求未被滿足，然後打造能加速可能性的事物。」

關鍵公司：OpenAI、Broadcom、Nvidia、Google TPU、Amazon Trainium

Claude Code 創辦人 Boris Cherny：AI Agent 自我迭代的「Loop」時代正式來臨

TechCrunch｜2026-06-22 20:53（UTC）｜✍️ Russell Brandom

Anthropic Claude Code 創辦人 Boris Cherny 在 Meta @Scale 大會上宣布，AI 發展正進入所謂「Loop」階段——多個 AI Agent 在幕後持續自我迭代、無限運行。他透露自己工作中有一個 Agent 持續尋找改進程式碼架構的方法，另一個則不斷搜尋可統一的重複抽象概念，兩者不斷提交 Pull Request。這些 Agent 會持續運行直到耗盡算力，而非傳統的明確終止條件。

Loop 的出現代表 AI 從「按需求回應」進化為「持續自主工作」。Cherny 比喻：「兩年前我們手寫程式碼，然後轉向 Agent 撰寫程式碼，現在則是 Agent 驅動 Agent。這是與從手寫到 Agent 同等重要的巨大跨越。」但代價是驚人的算力消耗。
