AI 晶片新創 Groq 確認完成 6.5 億美元融資：被 Nvidia 挖走創辦人後浴火重生

2026-06-22
AI 晶片新創 Groq 正式確認完成新一輪 6.5 億美元募資，由創投公司 Disruptive 與對沖基金 Infinitum 聯合領投，距 Nvidia 以非獨家授權模式取得 Groq LPU（語言處理器）IP 並挖走創辦人 Jonathan Ross 已約六個月。Groq 估值上一次揭露為 69 億美元（2025 年 9 月），新輪估值未公布。Ross 在 Google 時期共同打造了 TPU，後與 Doug Wightman 共同創辦 Groq；Wightman 在交易後接任 CEO。Nvidia 已將 LPU 整合進自家硬體叢集「Nvidia Groq 3 LPX」，Groq 因此轉向 neocloud 業務，目前在北美、歐洲、中東、亞太地區擁有 13 座資料中心，每周處理數兆 tokens、服務超過 500 萬開發者。Groq 新任命 xAI 與 Meta 前高管 Alan Rice 為 COO，以及共同創辦人 Sinclair Schuler 為 CTO、Rakesh Malhotra 為 CPO。
為什麼重要：Nvidia 的「非收購式併購」模式已成為 AI 半導體產業的重要現象——透過授權與挖角快速取得關鍵 IP 而無需承擔收購整合風險。Groq 能在核心 IP 被移轉後迅速轉型並成功募得大筆資金，顯示 AI 推論（inference）市場需求之強勁，也為其他被類似方式「吸血」的目標公司提供了復出範本。

Nvidia 推出資料中心熱水冷卻系統：號稱節省幾乎 100% 用水，但專家批評只解決了冰山一角

2026-06-22
Nvidia 宣布推出 45°C 熱水冷卻系統，透過閉路循環設計，號稱可在大多數氣候條件下不使用蒸發式冷卻或風扇，大幅減少資料中心內部用水——環保長 Josh Parker 宣称「資料中心的用水挑戰已基本解決」。然而批評者指出，Nvidia 的測量邊界僅計算資料中心內部用水，而發電廠（尤其化石燃料電廠）才是最大用水戶：燃氣電廠每發一度電消耗 1.17 公升水、燃煤更高達 2.2 公升；目前約一半資料中心電力仍來自化石燃料，整體用水足跡是設施內部的 2 至 3 倍。

AI 世界正在進入「迴圈時代」：Agent 驅動 Agent 自我迭代，Anthropic Boris Cherny 称其為真正的下一波

2026-06-22
Anthropic Claude Code 負責人 Boris Cherny 在 Meta @Scale 大會上表示，AI 發展正在進入第三階段：從「人類寫程式」（Source Code）到「Agent 寫程式」（Agents），現在是「Agent 持續提醒 Agent 寫程式」（Loops）。在這個模式下，一個 subagent 會不斷檢查程式碼架構是否有改善空間、另一個 subagent 負責尋找可合併的重複抽象，三者輪流對話並自動提出 PR——任務永遠不會真正結束。Cherny 強調：「迴圈與從人類寫 code 到 Agent 寫 code 這一步同等重要。」這個模式對 AI 的信任要求極高，但伴隨模型能力快速提升，可能成為真正接管實際工程工作的關鍵一步。
為什麼重要：Loops 的出現代表 AI 系統正從「服從指令」進化為「自主驅動、持續運作的數位組織」。若 Agent 能在後台自行改善程式碼品質，工程師的角色將從「親自寫 code」轉變為「設定目標與品質標準」。這對軟體產業的未來工作型態與 AI Agent 平台的中期商業模式都有深遠影響。
